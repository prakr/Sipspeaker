running the project:
   

Add the code attached to Netbeans and run.

to run the code from command line , go to IK2213_Sipspeaker/build/ and issue the command:

java -cp /classes/ ik2213_sipspeaker.Sipspeaker [-c config_file_name] [-user sip_uri] [-http http_bind_address] 

open a web browser and type http://localhost:4711/index.html or http://localhost:4711/ on the location bar to obtain the
 html page where to view/edit the current message to be played.
